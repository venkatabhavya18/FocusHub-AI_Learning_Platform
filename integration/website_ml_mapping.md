# Website to Machine Learning Mapping - FocusHub
This document explains how user interactions on the FocusHub website are mapped to machine learning models to generate personalized, cognitive learning outcomes.

## Learn Page -> Focus Score & Memory Decay
### User Interaction Data
- Video watch duration
- Pause and skip frequnecy
- Idle time
- Tab Switching behavior
### ML Models Used
- Focus Score Model
- Memory Decay Prediction Model
### Output
- Focus score percentage
- Memory strength estimation
- Suggested revision timing

## Quiz Page -> Evaluation & Recommendation
### User Interaction data
- Quiz Score
- Time taken per question
- Number of attempts
### ML Models Used
- Evaluation Module
- Recommendation Engine
### Output
- Performance level
- Weak concept identification
- Next learning recommendation

## Analytics Page -> Learning Insights
### Input
- Focus score history
- Memory decay predictions
- Quiz performance data
### Output
- Learning trend visualization
- Progress analytics
- Improvement insights

## Recommendation Page -> Personalized Learning Path
### Input
- Learner history
- Focus and memory analysis
- Performance evaluation
### Output
- Recommended videos
- Adaptive quiz difficulty
- Revision schedule

## Security Layer -> Anomaly Detection(Conceptual)
### Input
- Login patterns
- Session behaviour
### ML Usage
- Detection of suspicious activity
- Secure learning continuity
