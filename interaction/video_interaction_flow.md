# Video Interaction Flow - FocusHub

## Learning Session Start
- Student selects a verified learning video.
- System initializes a learning session.

## Tracked Interactions
- Total watch time
- Pause frequency
- Rewind actions
- Skip events
- Session idle time

## Behavioral Purpose
These interactions reflect:
- Attention span
- Engagement depth
- Learning seriousness

## ML Usage
- Focus Score computation
- Memory decay estimation
- Recommendation input

Videos act as stimuli ; the dataset is generated from user interaction.
