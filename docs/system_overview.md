# FocusHub - System Overview

## Purpose
FocusHub is built to help students stay focused while studying and develop productive learning habits using AI and Machine Learning.

## What the system does
- Tracks how student study.
- Analyze focus and learning patterns.
- Suggests what to study next.
- Helps students revise before they forget.

## Main Parts of the system

### 1.User Interface
- Study videos.
- Quizzes.
- Progress and streak display.

### 2.Data Collection
- Time spent studying.
- Quiz scores.
- Revision history.
- Focus behaviour(idle time , tab switching)
  **No camera or personal biometric data is used**

### Intelligence layer
This layer uses ML to:
- Calculate focus score.
- Predict memory decay.
- Recommend content.
- Adjust quiz difficulty.

### Storage
Stores:
- User data.
- Learning progress.
- Quiz results.
