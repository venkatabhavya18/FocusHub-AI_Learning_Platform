## FocusHub - Phase 5 : Model Implementation
### Objective
   This phase focuses on implementing the ML models designed in Phase 4.

### Models Implemented
- Random Forest Regressor
- Decision Tree Regressor
- K-Means Clustering
- Collaborative Filtering
- Recommendation Engine

### Dataset Used
student_engagement_dataset.csv

### Feature Set Used
- watch_time
- quiz_score
- completion_rate
- engagement_score

### Training Approach
- 80:20 train-test split
- R² score for regression evaluation
- Clustering for learner segmentation

### Output
The training pipeline executes all models sequentially and prints performance metrics.
